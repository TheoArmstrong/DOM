DOM e Formularios

DOM

DOM - Data Object Model é uma API(Application Programming Interface) que visa simplificar a forma de acessar a estrutura de um documento HTML. Facilitando criar, modificar, adicionar, retirar e manipular elementos HTML.

Formulários

São elementos HTML que possibilita a interação do usuário com a página web. Como exemplo temos caixas de texto, botões, caixas de seleção, entre outros. As tags form, label, input são tags do elementos formularios.

Exemplo

Construa uma interface que sejam digitados dois numeros e somados.
